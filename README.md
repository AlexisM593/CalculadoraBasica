#  Calculadora Básica en C

Este proyecto muestra cómo crear, compilar y ejecutar una **calculadora sencilla en lenguaje C** desde la terminal de Ubuntu.  
Permite realizar operaciones de **suma, resta, multiplicación y división**.

---

##  Cómo ejecutar el programa

### 1 Clonar el repositorio
```bash
git clone https://github.com/AlexisM593/CalculadoraBasica.git
cd CalculadoraBasica
```
### 2 Compilar el programa
```bash
gcc calculadora.c -o calculadora
```
### 3 Ejecutar la calculadora
```bash
./calculadora
```
